	Для запуска необходимо расположить текстовые файлы cars, motorcycles, RussianOutlets, Sales в папке с .exe файлом (по умолчанию ..\din\debug\).
	В текстовых файлах хранятся базовые значения для генерации случайных данных из представленного списка. Каждый вид данных обязательно указываются в своей строке через запятую, в конце запятая не ставится. Если необходимо добавить новые наименования, то нужно открыть вышеуказанные txt файлы в блокноте, добавить новые позиции в строку с аналогичными данными через запятую.
